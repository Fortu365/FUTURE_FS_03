# 🥩 Premium Cuts Butchery Website

## 📌 Project Overview

This project is a **professional website for a local butchery business**.
It was developed as part of a practical assignment to demonstrate real-world web development skills and business problem-solving.

The website helps the business:

* Promote products online
* Attract more customers
* Provide easy contact and ordering options
* Improve overall digital presence

---

## 🎯 Objectives

* Build a clean and responsive business website
* Showcase products and services
* Provide clear contact and ordering options
* Create a user-friendly experience for customers

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (basic interactions)
* Unsplash (for images)

---

## 🌐 Features

* ✔ Modern and responsive design
* ✔ Hero section with call-to-action
* ✔ Product showcase (Beef, Chicken, Lamb, Sausages, Braai Packs)
* ✔ Image gallery
* ✔ About section
* ✔ Customer testimonials
* ✔ Contact section with:

  * Location
  * Phone number
  * Business hours
  * WhatsApp order button
